# Images Folder

Place the following images in this folder:

## Required Images:

1. **adobe-logo.svg** - Adobe logo for the header
2. **avatar.png** - User profile avatar (the colorful circular icon)
3. **creative-cloud-sale.png** - The "SALE" promotional image with colorful creative elements
4. **creative-cloud-card.png** - Creative Cloud card with gradient background
5. **acrobat-card.png** - Adobe Acrobat marketing card with red/pink gradient

## Recommendations:

- Use SVG format for logos and icons when possible
- Use PNG for photographs and complex graphics
- Optimize images for web (compress before adding)
- Recommended avatar size: 200x200px
- Recommended card images: 800x400px
