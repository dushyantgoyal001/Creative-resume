# Icons Folder

Place the following icons in this folder:

## Social Sign-in Icons:
- **google.svg** - Google logo
- **facebook.svg** - Facebook logo
- **apple.svg** - Apple logo
- **microsoft.svg** - Microsoft logo
- **icloud.svg** - iCloud logo
- **message.svg** - Message/SMS icon

## Adobe Apps/Services Icons:
- **premiere-rush.svg** - Premiere Rush icon (Ru logo)
- **creative-cloud.svg** - Creative Cloud icon
- **lightroom.svg** - Lightroom icon (Lr logo)
- **photoshop-express.svg** - Photoshop Express icon (Ps logo)
- **fresco.svg** - Fresco icon (Fr logo)
- **behance.svg** - Behance icon (Be logo)
- **pdf.svg** - PDF Services icon

## Recommendations:

- All icons should be in SVG format for scalability
- Recommended size: 24x24px or 32x32px
- Use official brand icons when possible
- Ensure icons have transparent backgrounds
- Adobe product icons typically use their signature square design with rounded corners
